Case Study: Automação de Admissões

Python Obsidian Integração de IA Google Docs API

Data Pipeline: Obsidian → Google Docs

> O Contexto

O fluxo de admissões gerava gargalos absurdos e trabalho manual repetitivo que limitava o potencial estratégico e humano da nossa equipe de Infraestrutura.

> A Solução Desenvolvida

Fugindo do padrão de "apenas hardware", desenvolvi um script autônomo em Python. Este script consome arquivos brutos do Obsidian, processa a inteligência das informações (inspirado pelos modelos lógicos do NotebookLM) e injeta tudo de forma estruturada em relatórios oficiais no Google Docs.

> Impacto na Operação

O que antes era cópia manual demorada tornou-se um pipeline 100% autônomo. Isolamento dos dados com segurança no histórico, sem atrito, liberando mais tempo para inovar de verdade.
